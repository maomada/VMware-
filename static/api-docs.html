🔐 认证接口

POST /api/auth/login 公开
用户登录，获取访问令牌

请求参数

参数名 类型 必填 说明
username string 是 LDAP用户名
password string 是 用户密码

请求示例

{
  "username": "john.doe",
  "password": "your_password"
}

响应示例

{
  "success": true,
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "user": {
    "id": 1,
    "username": "john.doe",
    "display_name": "John Doe",
    "email": "john.doe@company.com",
    "department": "IT"
  }
}
POST /api/auth/logout
🔒 需要认证
用户登出，使令牌失效
POST /api/auth/verify
🔒 需要认证
验证令牌有效性

💻 虚拟机管理

GET /api/vms
🔒 需要认证
获取当前租户的虚拟机列表

查询参数

参数名 类型 必填 说明
project_id integer 否 过滤指定项目的虚拟机
POST /api/vms
🔒 需要认证
创建新的虚拟机

请求参数

参数名 类型 必填 说明
name string 是 虚拟机名称
template_name string 是 操作系统模板
cpu_cores integer 是 CPU核数
memory_gb integer 是 内存大小(GB)
disk_gb integer 是 磁盘大小(GB)
deadline string 是 过期时间(ISO格式)
POST /api/vms/{vm_id}/power/{action}
🔒 需要认证
虚拟机电源操作

action参数: on | off | restart

DELETE /api/vms/{vm_id}
🔒 需要认证
删除虚拟机

🔑 认证说明

大部分API需要在请求头中包含认证令牌：

Authorization: Bearer YOUR_JWT_TOKEN

错误码说明

状态码 说明
200 请求成功
400 请求参数错误
401 未认证或认证失败
403 权限不足
404 资源不存在
500 服务器内部错误
